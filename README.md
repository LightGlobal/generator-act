# generator-act 

> [Yeoman](http://yeoman.io) generator


## 安装

```
$ npm install -g yo
$ npm install -g generator-act
```

## 使用

### 生成项目

```
$ mkdir my-project
$ cd my-project
$ yo act
```

### 添加组件

```
$ yo act:cmp <cmp-name>
```

其中 `cmp-name` 为组件名

### 添加语言包

```
$ yo act:lang <language>
```

其中 `language` 为语言名称，如 `en`，`ru` 等
